A **Demultiplexer** is a combinational circuit that takes one input line and distributes it to one of several output lines, as determined by a set of select inputs.
Also known as a data distributor, the output lines are ‘n’ in number, the select line number is ‘m’ and n = 2^m. 
The control signal or select input code decides the output line to which the input has to be transmitted.
You can find more details about a Demultiplexer [here](https://www.tutorialspoint.com/digital-electronics/digital-electronics-demultiplexers.htm) .
